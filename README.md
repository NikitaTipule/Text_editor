# Text_editor

A C based text editor built with Ncurses library.

Data Structure used - buffer, linked list

## Usage



```bash
make 
./editor
```

## Features
    CREATING FILES
    SAVING FILES
    READING FROM FILES
    EDITING FILES
    CUT/COPY WORDS
    CUT/COPY LINES
    SEARCH WORDS
    REPLACE WORDS
    GO TO LINE NUMBER
    KEYBOARD MODE
    NICE UI INTERFACE
